# Icon 图标

> 图标

Yogurt UI 并不直接提供 icon 能力，你可以自由选择 icon 库，比如：

- [xicons](https://xicons.org/)

- [tdesign-icons-react](https://tdesign.tencent.com/react/components/icon)

- [@ant-design/icons](https://ant-design.antgroup.com/components/icon-cn)

## 使用 `xicons`

1. 安装 utils

```bash
pnpm add -D @ricons/utils 
```

2. 安装需要的组件包

```bash
pnpm add -D @ricons/fluent
pnpm add -D @ricons/ionicons4
pnpm add -D @ricons/ionicons5
pnpm add -D @ricons/antd
pnpm add -D @ricons/material
pnpm add -D @ricons/fa # font awesome
pnpm add -D @ricons/tabler
pnpm add -D @ricons/carbon
```
